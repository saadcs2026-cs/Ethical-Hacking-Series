# EP 30 — Setting Up Your Hacking Lab

## What is a Hacking Lab?
A safe isolated environment
where you practice hacking legally.

## Why You Need One
- Practice without breaking laws
- Test tools safely
- Make mistakes without consequences
- Build real skills before real work
- No risk to real systems

## What You Need
### Hardware
- Any laptop or PC
- Minimum 8GB RAM recommended
- 50GB free storage minimum

### Software (All Free)
| Software | Purpose | Download |
|----------|---------|----------|
| VirtualBox | Virtual machine | virtualbox.org |
| Kali Linux | Attack machine | kali.org |
| Metasploitable2 | Vulnerable target | sourceforge.net |
| DVWA | Vulnerable web app | github.com/digininja/DVWA |

## Step by Step Setup
Step 1 → Download VirtualBox
Step 2 → Download Kali Linux ISO
Step 3 → Create new VM in VirtualBox
Step 4 → Install Kali Linux on VM
Step 5 → Download Metasploitable2
Step 6 → Run both VMs together
Step 7 → Start practicing legally!

Kali = your attack machine
Metasploitable = your target

## Network Setup for Lab
- Set both VMs to Host Only Network
- They can talk to each other
- Cannot reach real internet
- Safe and isolated environment!
