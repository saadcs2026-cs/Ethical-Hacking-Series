# EP 22 — What is Kali Linux?

## Definition
Kali Linux is a Linux distribution
built specifically for ethical hacking
and penetration testing.

## Key Facts
- Developed by Offensive Security
- Based on Debian Linux
- Comes with 600+ pre-installed tools
- Completely free to download
- Updated regularly
- Download at: kali.org

## Pre-installed Tools
| Tool | Purpose |
|------|---------|
| Nmap | Network scanner |
| Metasploit | Exploitation framework |
| Wireshark | Packet analyser |
| Burp Suite | Web app testing |
| John the Ripper | Password cracker |
| Aircrack-ng | WiFi security |
| SQLmap | SQL injection |
| Hydra | Brute force |

## How to Get Kali Linux

### Option 1 — Virtual Machine (Recommended)
- Download VirtualBox — free
- Download Kali ISO from kali.org
- Run alongside Windows safely
- Best option for beginners!

### Option 2 — USB Boot
- Install on USB drive
- Boot from USB when needed
- No permanent installation

### Option 3 — WSL2
- Run Kali inside Windows
- No virtual machine needed
- Limited but convenient
