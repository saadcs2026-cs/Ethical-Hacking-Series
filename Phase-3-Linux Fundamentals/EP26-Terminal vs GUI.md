# EP 26 — Terminal vs GUI

## What is a GUI?
Graphical User Interface
- Visual interface with icons and buttons
- Easy for beginners
- Slow for advanced tasks
- Uses more system resources
- Not always available on servers

Examples: Windows Desktop, Mac OS,
Ubuntu Desktop

## What is a Terminal?
Command Line Interface (CLI)
- Type commands to control system
- Faster than GUI
- Uses minimal resources
- Works on all Linux systems
- Works on remote servers via SSH
- Full control of the system
- Essential for ethical hacking!

## Comparison
| Feature | Terminal | GUI |
|---------|----------|-----|
| Speed | Fast ✅ | Slow ❌ |
| Resources | Low ✅ | High ❌ |
| Control | Full ✅ | Limited ❌ |
| Remote use | Yes ✅ | Hard ❌ |
| Automation | Yes ✅ | No ❌ |
| Learning curve | Hard ❌ | Easy ✅ |

## Real World Fact
Most servers and hacking targets
have NO graphical interface.
Terminal is the only way
to interact with them!

## Common Terminal Emulators
- GNOME Terminal — Ubuntu default
- Konsole — KDE default
- Terminator — multiple panes
- Tmux — terminal multiplexer
