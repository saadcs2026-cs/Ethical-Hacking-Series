# EP 21 — Why Hackers Use Linux

## Main Reasons

### Open Source
- Anyone can view and modify the code
- Security researchers trust it completely
- No hidden backdoors like proprietary OS

### Full Control
- Access every part of the system
- No restrictions like Windows
- Modify anything you want

### Free Forever
- No license fees
- No activation keys
- No subscriptions

### Built in Networking Tools
- ifconfig, netstat, nmap
- All pre-installed
- Essential for hacking

### Server Dominance
- 90% of servers run Linux
- Understanding Linux means
  understanding your targets

### Privacy
- No background telemetry
- No data sent to Microsoft
- Complete anonymity possible

## Comparison
| Feature | Linux | Windows |

## Real World Fact
90% of the world's servers run Linux.
Hacking targets are mostly Linux systems.
You must know Linux to hack Linux!
