# EP 20 — Introduction to Linux

## What is Linux?
Linux is a free and open source
operating system kernel created by
Linus Torvalds in 1991.

## Who Created It?
**Linus Torvalds**
- Finnish software engineer
- Created Linux at age 21
- Originally just a hobby project
- Released it free to the world
- Still maintains the kernel today!

## Linux vs Windows vs Mac

| Customizable | ✅    | ❌      | ❌  |

## What is a Distribution (Distro)?
Linux is just the kernel — the core.
Distributions package it with
extra software and a user interface.

Kernel = car engine
Distribution = the full car

## Popular Distros
| Distro | Best For |
|--------|----------|
| Kali Linux | Ethical hacking |
| Ubuntu | Beginners |
| Debian | Servers |
| Parrot OS | Low end hardware |
| Arch Linux | Advanced users |
| CentOS | Enterprise |

## Which Distro to Use?
- Just starting Linux? → Ubuntu
- Learning ethical hacking? → Kali Linux
- Low end PC? → Parrot OS
- Deep Linux understanding? → Arch Linux
- For this series → Kali Linux ✅

- NASA uses Linux for space missions
- International Space Station runs Linux
