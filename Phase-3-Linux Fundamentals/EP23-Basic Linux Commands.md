# EP 23 — Basic Linux Commands

## Navigation Commands
| Command | What it Does |
|---------|-------------|
| pwd | Shows current directory |
| ls | Lists files and folders |
| ls -la | Shows hidden files too |
| cd Desktop | Move to Desktop folder |
| cd .. | Go back one folder |
| cd ~ | Go to home directory |
| clear | Clears the terminal |

## File Management Commands
| Command | What it Does |
|---------|-------------|
| touch file.txt | Create new file |
| mkdir folder | Create new folder |
| cp file.txt /tmp | Copy file |
| mv file.txt /tmp | Move file |
| rm file.txt | Delete file |
| rm -rf folder | Delete folder |
| cat file.txt | Read file content |

## Network Commands
| Command | What it Does |
|---------|-------------|
| ifconfig | Shows IP and network info |
| ip a | Modern version of ifconfig |
| ping google.com | Test connectivity |
| netstat -an | Shows open connections |
| nmap target | Scans network and ports |
| wget URL | Download from internet |
| curl URL | Fetch web content |

## System Commands
| Command | What it Does |
|---------|-------------|
| whoami | Shows current user |
| uname -a | Shows system info |
| ps aux | Shows running processes |
| kill PID | Stops a process |
| sudo command | Run as administrator |
| apt update | Update package list |
| apt install | Install new software |

## Pro Tips
- Tab key autocompletes commands
- Up arrow shows previous commands
- Ctrl+C stops running command
- Ctrl+L clears terminal
