# EP 19 — Packet Sniffing

## Definition
Capturing and analysing data packets
travelling across a network.

## How it Works
Data travels in packets
↓
Sniffer captures all packets
↓
Analyses content of each packet
↓
Unencrypted data → readable
Encrypted data → cannot read!

## This is Why HTTPS Matters!
HTTP → data visible to sniffer
HTTPS → data encrypted and safe

## How to Protect Yourself
- Always use HTTPS websites
- Use VPN on public WiFi
- Avoid HTTP sites for sensitive data
- Use encrypted messaging apps
- Never use public WiFi for banking

## Tools Used
- Wireshark — most popular
- tcpdump — command line tool
- Ettercap — network analysis
