# EP 14 — What is a MAC Address?

## Definition
MAC = Media Access Control
A unique physical address burned
into every network device.
Example: 00:1A:2B:3C:4D:5E

## Key Facts
- Never changes — burned into hardware
- Used on local network only
- Like a passport number for your device
- Every phone, laptop, router has one

## IP vs MAC
| | IP Address | MAC Address |
|-|------------|-------------|
| Changes | Yes | No |
| Used for | Internet | Local network |
| Like | Home address | Passport number |

## MAC Spoofing
Changing your MAC to pretend
to be a different device.
- Bypass network restrictions
- Hide real identity
- Used in penetration testing
