# EP 8 — What is DNS?

## Definition
DNS stands for Domain Name System.
The phonebook of the internet!
Converts website names into IP addresses.

## How it Works
Step 1 — You type google.com
Step 2 — Device asks DNS "What is the IP?"
Step 3 — DNS replies 142.250.190.46
Step 4 — Browser connects to that IP
Step 5 — Google loads on your screen!
All in milliseconds!

## DNS and Hacking
DNS Spoofing:
Hacker gives wrong IP
You type bank.com → fake bank site!

DNS Hijacking:
Hacker controls your DNS
All web traffic goes through them!
