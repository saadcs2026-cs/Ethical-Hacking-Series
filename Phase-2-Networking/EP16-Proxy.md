# EP 16 — What is a Proxy?

## Definition
A proxy server acts as a middleman
between your device and the internet.

## What it Does
- Hides your real IP address
- Websites see proxy IP not yours
- Filters and monitors traffic
- Used for privacy and anonymity

## Proxy vs VPN
| | Proxy | VPN |
|-|-------|-----|
| Hides IP | Yes | Yes |
| Encryption | No | Yes |
| Speed | Faster | Slower |
| Security | Basic | Strong |

## Types of Proxies
- Forward Proxy — hides client
- Reverse Proxy — hides server
- Transparent — does not hide IP
- Anonymous — hides your IP

## Hacker Use
Hackers chain multiple proxies
to stay completely anonymous.
Called proxy chaining!
