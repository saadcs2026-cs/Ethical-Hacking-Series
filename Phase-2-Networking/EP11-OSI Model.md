# EP 11 — The OSI Model

## What is the OSI Model?
OSI stands for Open Systems Interconnection.
A framework explaining how data travels
across a network in 7 layers.

## The 7 Layers
| Layer | Name | What it does |
|-------|------|--------------|

| 7 | Application | What you see — browser, email |
| 6 | Presentation | Encryption and formatting |
| 5 | Session | Opens and closes connections |
| 4 | Transport | TCP and UDP — splits data |
| 3 | Network | IP addresses and routing |
| 2 | Data Link | MAC addresses and frames |
| 1 | Physical | Cables and hardware |

## Hacker Perspective
- Phishing → Layer 7
- SSL attacks → Layer 6
- Session hijacking → Layer 5
- DDoS → Layer 4
- IP Spoofing → Layer 3
- MAC Spoofing → Layer 2
- Physical attacks → Layer 1
