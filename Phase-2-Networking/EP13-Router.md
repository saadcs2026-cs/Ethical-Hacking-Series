# EP 13 — What is a Router?

## Definition
A router connects your home network
to the internet and directs traffic
between devices.

## What it Does
- Gives every device an IP address
- Decides fastest path for data
- Acts as traffic controller
- Your WiFi signal comes from here

## How it Works
Device sends request
↓
Router receives it
↓
Router finds best path
↓
Sends to internet
↓
Receives response
↓
Sends back to your device

## Router vs Switch
| Device | Purpose |
|--------|---------|
| Router | Connects different networks |
| Switch | Connects devices within network |

## Security Risk
- Default passwords are easy to crack
- Control router = control all traffic
- Always change default password!
