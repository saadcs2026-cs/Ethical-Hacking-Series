# EP 18 — What is Wireshark?

## Definition
A free network protocol analyzer
that captures and inspects all
data packets on a network.

## What Wireshark Shows
- Every website you visit
- Every login request made
- Unencrypted passwords on HTTP
- DNS requests from all devices
- All devices on your network
- Suspicious traffic and attacks

## How Ethical Hackers Use It
- Analyse network vulnerabilities
- Detect suspicious activity
- Find unencrypted sensitive data
- Troubleshoot network problems
- Learn how protocols work

## Download
wireshark.org — completely free
Available on Windows, Mac, Linux

## Important Rule
Only use on networks you own
or have permission to analyse!
