# EP 7 — IP Addresses Explained

## Definition
IP stands for Internet Protocol.
A unique number assigned to every device.
Like a home address but for your device!

## Types of IP
Public IP:
- Your address on the internet
- Websites can see this!
- Example: 103.45.67.89

Private IP:
- Your address on home network
- Only local devices see this!
- Example: 192.168.1.5

## IPv4 vs IPv6
IPv4 — old format
192.168.1.1
Only 4 billion addresses — almost used up!

IPv6 — new format
2001:0db8:85a3:0000
340 trillion trillion trillion addresses!
