# EP 4 — How the Internet Works

## Core Concept
The internet is a massive conversation 
between two types of computers:
- Client — your phone or laptop
- Server — powerful computer storing data

## The 4 Step Journey
When you type abc.com:
1. DNS Lookup — finds IP address
2. Request — sends data packet
3. Routing — packet travels fastest path
4. Response — server sends data back

## Packets
Files are broken into tiny packets.
Each packet has a To and From address.
They reassemble in the right order.

## What is The Cloud?
Not a real cloud — just someone else's 
powerful computer acting as a server!
