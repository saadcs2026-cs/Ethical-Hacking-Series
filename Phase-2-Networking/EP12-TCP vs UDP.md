# EP 12 — TCP vs UDP

## TCP — Transmission Control Protocol
- Reliable and safe
- Confirms every packet delivered
- Resends lost packets
- Slower but 100% accurate
- Used for: websites, emails, files

## UDP — User Datagram Protocol
- Fast but unreliable
- No delivery confirmation
- Lost packets not resent
- Much faster than TCP
- Used for: gaming, video calls, streaming

## Comparison
| Feature | TCP | UDP |
|---------|-----|-----|
| Reliable | ✅ Yes | ❌ No |
| Speed | Slower | Faster |
| Confirmation | Yes | No |
| Use case | Websites | Gaming |

## Simple Analogy
TCP = Registered mail — confirmed delivery
UDP = Shouting in a crowd — fast, not guaranteed
