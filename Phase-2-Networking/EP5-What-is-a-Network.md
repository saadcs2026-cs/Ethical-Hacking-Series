# EP 5 — What is a Network?

## Definition
A network is a group of devices connected 
together to share data and resources.

## Types of Networks
- LAN — Local Area Network
  Your home or office network
- WAN — Wide Area Network
  Connects cities and countries
- WiFi — Wireless Network
  No cables, uses radio waves
- VPN — Virtual Private Network
  Secure private network over internet

## Your Home Network
- Router — connects to internet
- WiFi — connects devices wirelessly
- Switch — connects wired devices
- All devices share one connection!
